# Paris to Nice Road Trip 2024

**Trip dates:** December 15-22, 2024

## Overview
This repository tracks the planning, itinerary, and blog content for my scenic drive from the Eiffel Tower in Paris to the Promenade des Anglais in Nice, with an overnight stop in Lyon.

- **Start:** Eiffel Tower, Paris (48.8584° N, 2.2945° E)
- **Stopover:** Lyon (overnight at OKKO Hotels Lyon Centre)
- - **Finish:** Promenade des Anglais, Nice (43.6951° N, 7.2619° E)

## Project Structure
- `trip-content` branch – drafts of blog posts before merging to `main`
- `itinerary.md` – detailed route and stopover information
- GitHub Issues – tasks for each blog post
